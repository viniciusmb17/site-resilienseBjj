
# Resilience BJJ – Guia de Instalação e Desenvolvimento

Siga os passos abaixo para clonar, instalar e rodar o projeto localmente em modo de desenvolvimento.

## Pré-requisitos

- **Node.js** (recomendado: versão 18 ou superior)
- **pnpm** – Gerenciador de pacotes ([Guia de instalação](https://pnpm.io/installation))

## Passo a Passo

### 1. Clone o repositório

```sh
git clone https://github.com/gabrielms96/site-resilienseBjj.git
cd site-resilienseBjj
```

### 2. Instale as dependências

```sh
pnpm install
```

### 3. Rode o servidor de desenvolvimento

```sh
pnpm dev
```

### 4. Acesse o projeto no navegador

Abra [http://localhost:5173](http://localhost:5173) para visualizar o site.

---

## Outras informações

- Para builds de produção, utilize `pnpm build`.
- Para mais detalhes, consulte a documentação dos scripts no `package.json`.

---

Pronto! O projeto estará rodando em modo de desenvolvimento.
